# Mapa mundial de comunidades

## Completado

- Lista comunidades activas desde `groups`.
- Búsqueda por nombre, iglesia, ciudad o descripción.
- Filtro por país.
- Conteo por país.
- Sugerencia de comunidad con información de reunión.
- Admin puede aprobar/rechazar sugerencias.
- Al aprobar, se crea un grupo activo.
- Script `qa:map`.

## QA

- `npm run qa:map`: `QA_MAP_OK`.

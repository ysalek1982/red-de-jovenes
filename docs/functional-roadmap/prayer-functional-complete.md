# Sala de oración comunitaria

## Completado

- Crear petición de oración.
- Categorías: familia, salud, estudios, trabajo, fe, agradecimiento, otro.
- Opción visual anónima.
- Marcar “Estoy orando” y quitar apoyo.
- Marcar petición propia como respondida.
- Testimonio breve de respuesta.
- Filtros: todas, activas, respondidas, mías, estoy orando.
- Reportar petición.
- KPIs reales de apoyos y peticiones activas.

## Seguridad

- RLS evita marcar peticiones ajenas como respondidas.
- RLS evita duplicar apoyo de oración.

## QA

- `npm run qa:prayer`: `QA_PRAYER_OK`.

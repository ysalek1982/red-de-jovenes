# Foros con la Palabra completos

## Completado

- Crear publicación con versículo y referencia opcional.
- Editar publicación propia.
- Eliminar publicación propia.
- Comentar publicaciones.
- Editar comentario propio.
- Eliminar comentario propio.
- Reaccionar con “Amén” y quitar reacción.
- Reportar publicación y comentario.
- Filtros: recientes, con versículo, más comentadas.
- Conteos de comentarios y reacciones.

## Seguridad

- RLS bloquea edición/eliminación de contenido ajeno.
- Admin puede moderar por políticas existentes.

## QA

- `npm run qa:forums`: `QA_FORUMS_OK`.

## Pendiente

- Paginación y búsqueda textual cuando aumente la comunidad.

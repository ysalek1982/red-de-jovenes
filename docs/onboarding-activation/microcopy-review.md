# Revision de microcopy de activacion

Fecha: 2026-05-21

## Objetivo

Hacer que los textos visibles para usuarios nuevos sean mas claros, calidos y accionables, evitando lenguaje tecnico o mensajes frios.

## Cambios aplicados

| Area | Ajuste |
| --- | --- |
| Home | Checklist y perfil usan lenguaje de camino, comunidad y primera huella. |
| Biblia | Ayuda contextual invita a empezar con versiculo del dia, Juan 3 o Salmos 23. |
| Foros | Empty state invita a compartir reflexion, pregunta o palabra de animo. |
| Oracion | Ayuda contextual explica pedir oracion o acompanar a alguien. |
| Juegos | Primer paso explica que el puntaje se guarda al terminar. |
| Mapa | Mensaje invita a buscar ciudad, unirse o sugerir comunidad. |
| Eventos | Empty state explica que se podra confirmar asistencia. |
| Mensajes | Empty state orienta a conversar con jovenes de comunidades o perfiles sugeridos. |
| Textos rotos | Se corrigieron textos visibles con acentos y nombre "Red de Jóvenes" en pantallas tocadas. |

## Criterios

- No se agregaron popups.
- No se usaron metricas ficticias.
- No se modificaron flujos criticos.
- Seguridad sigue fuera del menu visible.

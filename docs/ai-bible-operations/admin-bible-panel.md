# Panel Admin Biblia

## Funciones

El panel admin muestra:

- traducciones cargadas;
- cantidad de libros;
- cantidad de versiculos;
- estado de licencia;
- versiculos diarios recientes;
- prueba de versiculo aleatorio;
- gestion basica de versiculo diario por fecha.

## Restriccion

La carga completa de archivos biblicos sigue siendo proceso administrativo por script para mantener control de licencia, formato y trazabilidad.

## QA

- `npm run qa:admin-bible`
- Resultado esperado: `QA_ADMIN_BIBLE_OK`

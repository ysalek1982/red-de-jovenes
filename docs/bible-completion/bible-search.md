# Busqueda biblica

Objeto nuevo: `search_bible_verses`.

## Funciones

- Busca por palabra.
- Busca por frase.
- Busca por referencia como `Juan 3:16`.
- Permite limitar por traduccion.
- Permite limitar por libro.
- Devuelve referencia, texto y ranking.

## UI

`/app/biblia` incluye una caja "Buscar en la Biblia" y resultados con acciones:

- Guardar.
- Subrayar.
- Copiar.
- Explicar con IA.
- Compartir en Foros.

## Limitacion actual

Si no se carga corpus completo, la busqueda se limita a los versiculos importados.

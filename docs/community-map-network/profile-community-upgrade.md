# Perfil comunitario

Fecha: 2026-05-19

## Cambios aplicados

- El perfil muestra comunidades propias desde `group_members`.
- Se agrego CTA directo a "Encontrar comunidad".
- Se agrego resumen de actividad personal:
  - devocionales leidos;
  - oraciones apoyadas;
  - juegos completados;
  - puntos de juegos.
- Se mantiene la edicion de datos personales, avatar, ciudad, pais, iglesia, bio y preferencias.

## Privacidad

- La vista sigue siendo propia del usuario autenticado.
- No se agrego perfil publico abierto.
- No se exponen listas de miembros de otras comunidades.

## Datos usados

- `profiles`
- `notification_preferences`
- `group_members`
- `groups`
- `devotional_reads`
- `prayer_supports`
- `game_scores`

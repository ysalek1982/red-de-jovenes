# Oracion - Fase 37

- La sala de oracion usa `app-page` y elimina decoracion fija innecesaria.
- El formulario principal usa card compartida y selects tactiles.
- Los filtros pasan a chips horizontales para mobile.
- Las peticiones usan cards mas calidas, con badges de estado, categoria y apoyos.
- Los estados de error, exito y vacio usan patrones compartidos.
- Se mantiene el boton principal "Estoy orando" y el reporte discreto.

- No se modifico `prayerService`.
- Crear peticiones, apoyar, quitar apoyo, marcar respondida y reportar siguen intactos.

# Checklist Post Release - pilot-v1.1.0

Fecha: 2026-05-21

## Pruebas humanas

| Prueba | Estado | Observacion |
| --- | --- | --- |
| PWA fisica Android | PENDIENTE | Instalar desde Chrome Android y abrir desde icono. |
| PWA fisica Desktop | PENDIENTE | Instalar desde Chrome/Edge y abrir standalone. |
| Recovery real | PENDIENTE | Solicitar recuperacion, abrir email y cambiar contrasena. |
| Login admin | PENDIENTE | Confirmar acceso admin desde dispositivo real. |
| Feedback real | PENDIENTE | Enviar feedback desde "Ayudanos a mejorar". |
| Incidente real de prueba | PENDIENTE | Crear/cerrar incidente controlado desde admin. |
| Biblia lectura larga | PENDIENTE | Leer capitulo largo en mobile y desktop. |
| Juego movil | PENDIENTE | Completar un juego usando solo tactil. |
| Foro movil | PENDIENTE | Crear post/comentario desde mobile. |
| Oracion movil | PENDIENTE | Crear peticion y apoyar otra desde mobile. |

## Reglas de soporte

- No activar Gemini sin instruccion del admin.
- No cargar otro corpus biblico sin licencia clara.
- No reintroducir Seguridad como modulo visible.
- Registrar bugs reales en feedback/incidentes.
- Corregir solo bloqueos, privacidad, navegacion rota o UX critica durante piloto.

## Recomendacion

Ejecutar este checklist en las primeras 24 horas del piloto cerrado y actualizar el soporte diario con hallazgos reales.

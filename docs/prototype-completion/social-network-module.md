# Seguimiento social entre jovenes

## Objetivo

Hacer visible que hay jovenes conectados sin crear chat en tiempo real ni exponer datos sensibles.

## Funciones

- Sugerencias de perfiles desde datos reales.
- Seguir usuario.
- Dejar de seguir.
- RLS bloquea modificar relaciones ajenas.

## Datos

Tabla:
- `user_follows`

## QA

Script: `npm run qa:social`  
Resultado esperado: `QA_SOCIAL_OK`

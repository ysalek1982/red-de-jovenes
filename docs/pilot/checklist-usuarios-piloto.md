# Checklist para usuarios piloto

Usa esta lista durante la primera sesion guiada del piloto.

## Pasos

1. Abrir la app desde el enlace compartido por el equipo.
2. Crear una cuenta nueva.
3. Iniciar sesion.
4. Instalar la app en el dispositivo si aparece la opcion.
5. Completar el perfil: nombre, ciudad, pais, iglesia o comunidad y bio.
6. Crear una peticion en la Sala de oracion global.
7. Abrir otra peticion y marcar “Estoy orando”.
8. Publicar en Foros con la Palabra.
9. Comentar o reaccionar con “Amen” en una publicacion.
10. Leer el devocional diario.
11. Marcar el devocional como leido o guardarlo.
12. Jugar Versiculo Rapido o Trivia Biblica.
13. Revisar el Mapa mundial.
14. Sugerir una comunidad o grupo si corresponde.
15. Reportar contenido de prueba indicado por el equipo.
16. Cerrar sesion.
17. Enviar feedback usando el formulario del piloto.

## Resultado esperado

El usuario debe poder completar el recorrido sin ayuda tecnica permanente.

Registrar cualquier bloqueo en `docs/pilot/matriz-monitoreo-piloto.md`.

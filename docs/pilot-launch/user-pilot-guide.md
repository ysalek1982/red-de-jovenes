# Guia rapida para usuarios piloto

## Que es Red de Jovenes

Red de Jovenes es una comunidad cristiana para conectar, orar, leer la Biblia, compartir, jugar y crecer juntos en Cristo.

## Como empezar

1. Abre https://red-de-jovenes.vercel.app.
2. Crea tu cuenta.
3. Inicia sesion.
4. Completa tu perfil con nombre, ciudad, pais e iglesia/comunidad.

## Que probar

- **Biblia:** entra a Biblia, lee el versiculo del momento y guarda uno.
- **Oracion:** crea una peticion o acompana a alguien con “Estoy orando”.
- **Foros:** comparte una reflexion, pregunta o testimonio.
- **Juegos:** juega Trivia Biblica o Versiculo Rapido y guarda tu puntaje.
- **Mapa:** busca comunidades, unete a una o sugiere una nueva.
- **Eventos:** revisa eventos disponibles y confirma asistencia.
- **Mensajes:** inicia una conversacion que edifique.
- **Reportar:** si algo te preocupa, usa el boton de reportar.

## Instalar como app

En Android/Chrome o desktop compatible, abre el menu del navegador y elige instalar app. Luego abre Red de Jovenes desde el icono.

## Normas basicas

- Habla con respeto.
- No compartas datos sensibles.
- No acoses ni presiones a nadie.
- Reporta contenido inapropiado.
- Si algo te preocupa, avisa a un lider o adulto responsable.

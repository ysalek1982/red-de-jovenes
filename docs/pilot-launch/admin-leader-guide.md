# Guia rapida para administradores y lideres

## Acceso

1. Entrar en https://red-de-jovenes.vercel.app.
2. Iniciar sesion con cuenta admin o lider autorizado.
3. Abrir `/app/admin` si el rol lo permite.

## Tareas principales

- Revisar KPIs y estado del piloto.
- Revisar reportes pendientes.
- Cambiar estado de reportes.
- Aprobar o rechazar comunidades sugeridas.
- Crear eventos.
- Revisar actividad de oracion.
- Gestionar Biblia y versiculo diario.
- Configurar Gemini solo desde el panel admin.
- Revisar uso, limites y prompts IA.
- Aprobar/rechazar acciones IA cuando corresponda.

## Incidentes

- Reporte grave: revisar, documentar y escalar.
- Usuario en crisis: contactar lider adulto/profesional/servicios locales.
- IA inadecuada: no publicar, desactivar si hace falta y ajustar prompt.
- Key expuesta: desactivar Gemini, rotar key y revisar logs.

## Que no hacer

- No compartir keys.
- No publicar contenido IA sin revisar.
- No borrar datos sin respaldo o criterio claro.
- No usar IA como consejeria profesional.
- No exponer datos privados de usuarios.
